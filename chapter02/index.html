Return Values

These exercises correspond to this section of the text.

For these next exercises, use the following functions:

magicFunc() takes no parameters and returns a numerical value.

strShift( str ) takes a string and returns a new string with all the letters shifted down the alphabet one place.

strShuffle( str1, str2 ) takes two strings and returns a new string with all the letters shuffled together.

Exercise # Goal
Return 1 Call and log out the result of magicFunc.
Return 2 Call and return the result of magicFunc.
Return 3

Call strShift with the argument "abczABCZ". Log the result. Then return that same result with "123" concatenated to the end.

Hint: Save the result of strShift in a variable.
Return 4 Log the result of the string concatenation of two separate calls to magicFunc.
Return 5

Fill in the function provided to do the following:

  1. Shift the string abcABC123!@# with strShift.
  2. Shift the string a S d F j K l with strShift twice.
  3. Return the result of these two shifted strings passed into strShuffle with the first string as the first argument and the second string as the second argument.

Once you have your solution, refactor it to contain only three statements in which...

  1. Statement #1 declares and initializes a variable for the first string.
  2. Statement #2 declares and initializes a variable for the second string.
  3. Statement #3 is a return statement.

Further Instructions

The book says to create a string and test it with console.log(). When you've got it figured out, instead of logging the string out, return it.

When you have finished all the specifications in the book, modify your function to accept a parameter n that, if passed into your function, will dynamically specify the size of the chessboard to return. Remember you can use the logical OR operator to choose the first item that exists:

var size = n || 8;

If there is no n, n || 8 will default to 8.
